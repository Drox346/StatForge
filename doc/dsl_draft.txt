DSL cell formula:

()  =  Same as in math. Inner Expressions are evaluated first.
<name>  =  References a specific cell.
1.2  =  Hard coded constant.
x + y  =  math plus
x - y  =  math minus
x * y  =  math multiply
x / y  =  math divide
root(x, y)  =  xth root of y
x^y  =  x to the power of y
x ? y : x * 2  =  if x, then return y. else return x times 2
< > <= >= == !=  =  Comparison operators
&& ||  =  Combining comparisons

Examples:

[val] level: "47"
[val] subclass: "1"
[val] baselife: "150"
[val] flatLifePerLevel: "48"

[for] flatLifeSubclass: "<subclass> == 0 ? 0 : 
                        <subclass> == 1 ? 100 : 
                        <subclass> == 2 ? 300 : 0"
[for] flatLifeLevel: "<level> * <flatLifePerLevel>"

[agg] flatLifeAggr: 
dependency: baselife, flatLifeLevel, flatLifeSubclass

Notes: 
    - [val] = value cell. you dont set a formula. just a value
    - [for] = formula cell. you set a custom formula
    - [agg cell] = you dont set a formula. it simply adds all dependencies
